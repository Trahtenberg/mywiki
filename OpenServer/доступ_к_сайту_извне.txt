====== доступ к сайту извне ======
Created пятница 23 Ноябрь 2018


Условие:
	Есть OpenServer, в нем есть сайт
Задача:
	дать доступ из сетки (с другого компьютера в сети) на сайт
	
- Добавить домен: [[OpenServer]] -> настройки -> Домены: apsmaster.loc -> \apsmaster\frontend\web
(чтобы [[OpenServer]] знал где лежит index.php)

- Добавить алиас [[OpenServer]] -> настройки -> Алиасы: 192.168.75.137 -> apsmaster.loc
(чтобы комп знал куда направлять внешние запросы на порт 80)

- Поменять значение [[OpenServer]] настройки -> Сервер -> IP-адрес сервера -> на "Все доступные IP"
(чтобы [[OpenServer]] обрабатывал не только внутренние, но и внешние запросы)
