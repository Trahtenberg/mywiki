====== теория 14.08.2019 ======
Created среда 14 Август 2019

- PHP как и C, Perl требует окончания команды точкой с запятой, 
	**<?php echo 'Это тест'; ?>**
	
	- но, последнюю команду перед закрывающим тегом ?> можно не закрывать точкой с запятой, т.к. закрывающий тег сам делает эту функцию
		**<?php echo 'Это тест' ?>**
		
	- но, если закрывающего тега нет, то точка с запятой обязательна
		**<?php echo 'Мы опустили последний закрывающий тег';**

- комментарий есть 
	// ...
	[[/*]] ... */
	а еще есть
		# ...

продолжаем с https://www.php.net/manual/ru/langref.php
