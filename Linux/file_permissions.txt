====== file permissions ======
Created среда 28 Август 2019

**права доступа к файлам (file permissions)**

у файлов есть владельцы
владельцами могут выступать пользователи или группы пользователей

Права доступа: **чтение, запись и исполнение**

Право на **чтение**: читать содержимое файла или просматривать содержимое каталога (ls)

Право на **запись** **в файл**: записывать или изменять файл
Право на **запись в каталог**: возможность создавать новые файлы или удалять файлы

Право на **исполнение для файла**: позволяет пользователю запускать файл или скрипт как программу
Право на **исполнение для каталога**: позволяет перейти (командой cd) в этот каталог.

ls -l 
-rwxrwxrwx

r - read
w - write
x - execute

1-й символ: тип файла, "-" файл, "d" папка
2-4 символы: права доступа владельца
5-7 символы: права доступа группы
8-10 символы: права доступа других(остальных) пользователей

Изменение прав доступа

chmod {a,u,g,o}{+,-}{r,w,x} filenames
{a,u,g,o} - all,user,group,other
{+,-} - add or remove
{r,w,x} - read,write,execute

например,

chmod a+r file
