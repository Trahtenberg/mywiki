====== ссылки на файлы ======
Created среда 21 Август 2019

просмотр номер индексного дескриптора (inode):
ls -i

создание жесткой ссылки:
ln filename hardlink

создание символической ссылки:
ln -s filename linkname

при жесткой ссылке inode файла и ссылки одинаковы

если удалить файл на которую есть жесткая ссылка, то со ссылкой ничего не произойдет, она также будет показывать файл и файловой системы
(это потому что файл считается удаленным если на него нет ссылок)

а если удалить файл на которую есть символическая ссылка, то ссылка будет выдавать ошибку

просмотр числа ссылок на файл:
ls -l
