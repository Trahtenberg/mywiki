====== установка стека vue ======
Created пятница 29 Март 2019

-Установить на локале NODEJS
"node -v" в командной строке покажет установленную версию

-установить vue cli

-запуск сервера "vue ui"

-смотрим на "localhost:8000"

-добавить или создать проект
в проекте aps, например, добавлено в [[/frontend/assets/,]] почему там?
потому что:
	- результатом компиляции vue cli являются файлы .js, .css
	- а js и css файлы добавляются в проект yii2 при помощи ассетов
	- а все ассеты в проекте yii2, включая классы ассетов, находятся в [[/frontend/assets/,|/frontend/assets/]]
(конечно расположение может быть и по другому)
