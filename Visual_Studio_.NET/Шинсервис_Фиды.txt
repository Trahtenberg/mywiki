====== Шинсервис Фиды ======
Created вторник 03 Декабрь 2019

для запуска проекта необходимо задать параметы в консоль:

правая кнопка на названии проекта справа -> Свойства -> tab Отладка -> поле Аргументы приложения:

примеры:
1_1 beru-ru d:/temp/feed.xml
1 yama d:/temp/feed.xml

параметр 1.
только для беру 1-й параметр: idregion_idshop
для остальных: id_region

параметр 2.
для кого делается:
	yama - яндекс маркет
	beru-ru

параметр 3.
	создаваемый файл: d:/temp/feed.xml
