====== запуск .exe в консоли ======
Created пятница 29 Ноябрь 2019

- при запуске из VS автоматически открывается командная строка и все ок

- вручную можно перейти в папку с .exe и запустить

- запуск через Git Bash - находясь в папке с .exe написать [[./consoleapp1.exe]]
